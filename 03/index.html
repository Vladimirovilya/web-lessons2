Властивість font-size

Властивість font-size змінює розмір шрифта.
Розмір шрифту задається за допомогою значення та одиниць виміру (em, px, pt, %)

Синтаксис:

CSS

p{
    font-size: 20px;
}

HTML

<p> Hello, world! </p>

Результат:

Hello, world!

Властивість font-weight

Властивість font-weight встановлює насиченість шрифту.
Може приймати наступні значення:

  • normal - нормальна насиченість, те саме, що і 400
  • bold - напівжирний текст, те саме, що і 700
  • lighter - змінює насиченість відносно батьківскього стилю в меншу сторону
  • booker - змінює насиченість відносно батьківскього стилю в більшу сторону
  • 100,200,300,400,500,600,700,800,900 - цифрові значення насиченості, які відрізняються від стандартних

Синтаксис:

CSS

.bold{
    font-weight: bold;
}
.thin{
    font-weight: 100;
}
.max{
    font-weight: 900;
}

HTML

<p>
   <span class = "bold">Напівжирний текст</span>
   <span class = "thin">Найтонший текст</span>
   <span class = "max">Найтовший текст</span>
</p>

Результат:

Напівжирний текст, найтонший текст, найтовщий текст

Властивість font-style

Властивість font-style зображення шрифту – звичайне, курсивне або похило.
Може приймати наступні значення:

  • normal - звичайне зображення тексту.
  • italic - курсивне зображення.
  • oblique - похило накреслення.
  • inherit - наслідує значення батька.

Синтаксис:

CSS

.normal{
    font-style: normal;
}
.italic{
    font-style: italic;
}
.oblique{
    font-style: oblique;
}

HTML

<p>
   <span class = "normal">Звичайний текст</span>
   <span class = "italic">Курсивний текст</span>
   <span class = "oblique">Похилий текст</span>
</p>

Результат:

Звичайний текст, курсивний текст, похилий текст

Властивість font-family

Властивість font-family встановлює сімейство шрифту, яке використовуватиметься для оформлення тексту вмісту.
Може приймати наступні значення:

  • serif - шрифти із засічками (антиквенні), типу Times;
  • sans-serif - рубані шрифти (шрифти без засічок чи гротески), типовий представник - Arial;
  • cursive - курсивні шрифти;
  • fantasy - декоративні шрифти;

Синтаксис:

CSS

.serif{
    font-family: serif;
}
.sans-serif{
    font-family: sans-serif;
}
.cursive{
    font-family: cursive;
}
.fantasy{
    font-family: fantasy;
}

HTML

<p>
   <span class = "serif">Шрифти із засічками</span>
   <span class = "sans-serif">Рубаний шрифт</span>
   <span class = "cursive">Курсивний шрифт</span>
   <span class = "fantasy">Декоративний шрифт</span>
</p>

Результат:

Шрифти із засічками, рубаний шрифт, курсивний шрифт, декоративний шрифт,

Властивість color

Властивість color визначає колір тексту елемента.
Може приймати наступні значення:

  • inherit - наслідує значення батька.

Синтаксис:

CSS

.inherit{
    color: inherit;
}

HTML

<p>
   <span class = "inherit">Значення батька.</span>
</p>

Результат:

Значення батька

Властивість text-transform

Властивість text-transform керує перетворенням тексту елемента у великі або маленькі символи.
Може приймати наступні значення:

  • capitalize - перший символ кожного слова в реченні буде великим. Інші символи свого вигляду не змінюють.
  • lowercase - усі символи тексту стають малими (нижній регістр).
  • uppercase - усі символи тексту стають великими (верхній регістр).
  • none - не змінює регістр символів.
  • inherit - наслідує значення батька. ;

Синтаксис:

CSS

.capitalize{
    text-transform: capitalize;
}
.lowercase{
    text-transform: lowercase;
}
.uppercase{
    text-transform: uppercase;
}
.none{
    text-transform: none;
}
.inherit2{
    text-transform: inherit;
}

HTML

<p>
   <span class = "capitalize">Лише перший символ буде великим</span>
   <span class = "lowercase">Усі символи маленькі</span>
   <span class = "uppercase">Усі символи великі</span>
   <span class = "none">Не змінює регістр символів</span>
   <span class = "inherit2">Наслідує значення батька</span>
</p>

Результат:

Лише перший символ буде великим, усі символи маленькі, усі символи великі, не змінює регістр символів, наслідує значення батька,

Властивість text-decoration

Властивість text-decoration додає оформлення тексту як його підкреслення, перекреслення, лінії над текстом і миготіння.
Може приймати наступні значення:

  • blink - встановлює миготливий текст.
  • line-through - створює перекреслений текст.
  • overline - лінія проходить над текстом
  • underline - встановлює підкреслений текст.
  • none - скасовує всі ефекти, у тому числі підкреслення у посилань.

Синтаксис:

CSS

.blink{
    text-decoration: blink;
}
.line-through{
    text-decoration: line-through;
}
.overline{
    text-decoration: overline;
}
.underline{
    text-decoration: underline;
}
.none2{
    text-decoration: none;
}

HTML

<p>
   <span class = "blink">Миготливий текст</span>
   <span class = "line-through">Перекреслений текст</span>
   <span class = "overline">Лінія проходить над текстом</span>
   <span class = "underline">Підкреслений текст</span>
   <span class = "none2">Скасовує всі ефекти</span>
</p>

Результат:

Миготливий текст, перекреслений текст, лінія проходить над текстом, підкреслений текст, скасовує всі ефекти
